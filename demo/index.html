cyTree v0.1.0

Features

  • Based on AngularJs 1.3.x, maybe compatible with AngularJs 1.2.x(not tested).
  • No jQuery dependency.
  • Compatible with modern browsers.

What's great?

  • You can custom the tree template per cyTree directive via inserting template into cyTree(see example2). Besides, custom cyTreeUtils.genTemplate so you can custom template globally.
  • Flexable property name setting. It means no need to change your data.
  • A builtin method(cyTreeUtils.formatData) to tranform the jQuery pluginjstree data structure to fit our directive, thus no pain to change.

Example-1(standard)

Usage:
<div class="cy-tree with-line" cy-tree="true" tree-data="treeData" tree-label-flag="name" tree-child-flag="list" debug-mode="true">
</div>
Description:
  • Directive cyTree add default behaviour and handle data.
  • Directive treeData correspond with each subtree.
  • The class cy-tree provide the base style, and the class with-line add line indicator to the tree.

Example-2(custom template)

Usage:
<div class="cy-tree" cy-tree="true" tree-data="treeData2" tree-label-flag="name" tree-child-flag="list" debug-mode="true">
    <ul>
        <li ng-class="{'last-node': $last, 'has-child': node.list.length}" ng-repeat="node in treeData2">
            <i class="fa" ng-if="node.list.length" ng-click="treeMethods.collapseNode(node)" ng-class="{'fa-angle-right': node.collapsed, 'fa-angle-down': !node.collapsed}"></i>
            <i class="fa" ng-if="!node.list.length"></i>
            <i class="fa" ng-class="{'fa-square-o': node.nonSelected, 'fa-check-square': node.selected, 'fa-square': node.partSelected}" ng-click="treeMethods.selectNode(node)"></i>
            <span class="tree-label">{{node.name}}</span><div class="cy-sub-tree" ng-if="node.list.length" ng-hide="node.collapsed" tree-data="node.list"></div>
        </li>
    </ul>
</div>
