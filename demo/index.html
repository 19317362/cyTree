cyTree v0.0.1

Features

  • Based on AngularJs 1.3.x, maybe compatible with AngularJs 1.2.x(not tested).
  • Some style features are CSS3 only, but you can custom/replace it.
  • No jQuery dependency.

What's good?

  • You can custom template or use the builtin one.
  • More flexable property name setting.
  • A builtin utility to tranform the jQuery pluginjstree data structure to fit our directive, thus no pain to change.

Usage:

Description:

The class cy-tree provide the base style, and the class with-line add line indicator to the tree.
